# 🚀 Proxy-Hunter

**Proxy-Hunter** — это продвинутый инструмент для автоматического сбора, анализа и преобразования прокси-конфигураций из публичных Telegram-каналов. Проект не просто парсит ссылки — он оценивает их качество, фильтрует мусор и выдаёт готовый к использованию набор для Xray.

> [!IMPORTANT]
> Проект создан для личного использования и образовательных целей. Автор не несёт ответственности за использование полученных данных. Соблюдайте законодательство вашей страны.

---

## ✨ Ключевые возможности

- 📡 **Сбор конфигураций** из 100+ публичных Telegram-каналов
- 🔍 **Парсинг и валидация** URI-ссылок всех популярных протоколов
- 🧹 **Глубокая дедупликация** — удаление дубликатов на уровне сервер+порт+учётные данные
- ✅ **Проверка работоспособности** — TCP/TLS-коннект, HTTP-пробник, замер задержек
- 📈 **ML-экспорт** — данные о качестве для обучения моделей
- 📊 **Генерация отчётов** — интерактивный HTML-дашборд с метриками по каналам

---

## 🗂️ Структура проекта

| Путь | Назначение |
|------|------------|
| `assets/` | HTML-отчёты и визуализации |
| `configs/` | JSON / TXT файлы с конфигурациями, кэшем локаций, историей качества |
| `src/` | Исходный код на Python |
| `install.sh` | Скрипт автоматической установки |
| `run.sh` | Основной пайплайн (оптимизированная версия) |
| `requirements.txt` | Python-зависимости |
